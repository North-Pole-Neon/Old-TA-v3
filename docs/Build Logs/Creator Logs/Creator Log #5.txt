after completion. It can store and read data just fine

are a bit tricky so I'm holding off on them for a bit until 
I get back from break.

so I will have to wait to test it to see if it works.

Right now I'm going to focus on housekeeping on the back end 
of things to make things easier to write and use as well as 
work on a task list for the user.
